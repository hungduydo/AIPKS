# Chrome STT Extension

## Idea
A Chrome extension that uses the Web Speech API to transcribe speech to text on any webpage.

## Notes
- Use `webkitSpeechRecognition` for browser-native STT
- Content script injects a floating mic button
- Background service worker handles state

## Status
In progress - basic architecture defined.
